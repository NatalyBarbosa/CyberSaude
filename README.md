# CyberSaúde

O CyberSaúde é um projeto de extensão universitária desenvolvido pelo capítulo IEEE SMCS da Universidade Federal do Recôncavo da Bahia (UFRB). O projeto visa mitigar a lacuna de acesso a tecnologias de saúde preventiva em comunidades quilombolas, em Cruz das Almas (BA), utilizando a computação física como ferramenta de transformação social e equidade regional.

Este repositório documenta a implementação de protótipos de monitoramento de sinais vitais de baixo cust.

## 🩺 Protótipos Desenvolvidos
Os dispositivos foram projetados priorizando a portabilidade, autonomia energética e o uso de hardware de código aberto:

* Frequência Cardíaca e Saturação (SpO2): Sistema não invasivo baseado no sensor MAX30102 e microcontrolador ESP32.
* Termômetro Digital Sem Contato: Utiliza o sensor infravermelho MLX90614 (leitura de radiação) e sensor VL53L0X.
* Medidor de Pressão Arterial: Baseado em Arduino Uno, utiliza sensor piezoresistivo MPX2050DP e sistema de braçadeira inflável automatizada.

## 🛠️ Metodologia
O projeto adota uma abordagem pedagógica progressiva, organizada em quatro etapas:

1. Formação técnica: Estudos de eletricidade, eletrônica e instrumentação biomédica.
2. Implementação: Desenvolvimento das arquiteturas de microcontroladores com foco na viabilidade econômica.
3. Preparação Pedagógica: Elaboração de conteúdos de apoio para letramento tecnológico.
4. Atuação Extensionista: Socialização do conhecimento e validação participativa junto à comunidade quilombola.
